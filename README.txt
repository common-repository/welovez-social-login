== Description ==

WeLovez Social Login For WoWonder Php Social Script a free powerful WordPress plugin that will allow wowonder (your social media) users to login to any wordpress website using their WOWONDER social account.
If you have a social networking website based on WOWONDER PHP SCRIPT, Then you can offer this plugin to six million wordpress website owners to install and allow your social media users to access those wordpress blogs using your wowonder based social media account.
So all of your social network users can login to six million websites easily and quickly. A great plugin to promote your social media network.
Contact me so i issue one for your website branding.

Features:
** WordPress plugin to let your wowonder social media users connect to any WordPress blog using their WoWonder account
** This is a very simple plugin that registers two new shortcodes in WordPress: `[wowonder_login]` and `[wowonder_share]`
** Wordpress blog authors can place these shortcodes anywhere to allow wowonder users login quickly or share content to wowonder social media.
** which displays a button to let wowonder users register or login to your WordPress site using wowonder and share your posts to wowonder website.
** WeLovez Social Login For WoWonder Php Social Script is very simple and easy to setup. All the instructions also mentioned in WeLovez Social Settings Page (WP ADMIN > Left SideBar WeLovez SOCIAL)


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin in WordPress > Plugins > WeLovez Social Login > Activate
3. Set your APP ID and APP Secret from the left wordpress admin dashboard sidebar > WeLovez Social
4. Use the `[wowonder_login]` shortcode in any page to login or register using WoWonder API.
5. Use the `[wowonder_share]` shortcode in any page to share posts directly to WoWonder without leaving your website.
